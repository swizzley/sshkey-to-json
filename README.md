# sshkey-to-json
CLI tool to quickly parse and format an SSH key pair into a JSON object

## Install

`go get github.com/swizzley/sshkey-to-json`

`go install github.com/swizzley/sshkey-to-json`

## Usage

`sshkey-to-json -pub=_testdata/testKey.pub -pvt=_testdata/testKey`
